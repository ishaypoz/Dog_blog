RESTful Routes

A table of RESTful routes

Name Path Http verb Purpose Mongoose Method
Index /dogs GET List all dogs Dog.find()
New /dogs/new GET Show new dog form
Create /dogs POST Create a new dog, then redircet Dog.create()
Show /dogs GET Show info about specific dog Dog.findById()
Edit /dogs/:id/edit GET Show edit form for one dog Dog.findById()
Update /dogs/:id PUT Show edit form for one dog Dog.findByIdAndUpdate()
Destroy /dogs/:id DELETE Delete a particular dog, then redirect Dog.findByIdAndRemove()
